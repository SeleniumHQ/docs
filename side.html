Selenium-IDE

Introduction

The Selenium IDE(Integrated Development Environment) is a record/playback tool that is designed to offer the selenium API and to allow people to export to a programming language. It's an easy-to-use Web Extension, usable in Chrome, Firefox, and any browser that supports that standard. It records the users actions in the browser for you, using existing Selenium commands, with parameters defined by the context of that element.
